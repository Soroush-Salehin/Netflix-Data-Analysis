# Netflix-Data-Analysis

This project is an end-to-end data analysis pipeline using the Netflix dataset from Kaggle: 

https://www.kaggle.com/datasets/shivamb/netflix-shows

It demonstrates skills in data cleaning, database design, SQL transformation, and interactive dashboard creation.

Tools Used:

Excel : Data cleaning & preprocessing

MySQL Workbench : Database creation, normalization, SQL transformations

Power BI : Data modeling & dashboard visualization
